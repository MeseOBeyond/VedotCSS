# Vedot CSS 一个不能让UI设计看到的CSS样式
- VedtoCSS包含了数些让前端们抓狂的UI设计，你可以随意地使用它们，或者将他们二次封包成一个组件
- 这个项目本来是想用Vue+TypeScript构架的，但是为了让它更加的轻量、可修改性广，我还使用原生的HTML+CSS+JS来构架它
